# PHP MVC
Yet another php mvc micro framework

## Installation

- composer way:

```console
foo@bar:~$ composer create-project develhopper/php-mvc
```

- git way:

```console
foo@bar:~$ git clone https://github.com/develhopper/php-mvc.git
foo@bar:~$ cd php-mvc && composer install
```
